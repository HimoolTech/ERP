# 海鸥云ERP_PDA移动端

### 介绍
海鸥云ERP移动端，采用前后端分离技术，api使用restful协议，方便二次开发，后端使用Python，Django，DRF等技术，移动端使用uniapp进行构建，主要包含仓库操作人员的扫码操作流程
